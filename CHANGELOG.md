# Changelog

All notable changes to MurlokFury will be documented in this file.

## [1.0.0] - 2026-02-08

### Added

- Initial release targeting WoW 12.0.0 (Midnight)
- Display murlok.io Fury Warrior M+ meta data in-game
- 6-tab interface: Talents, Stats, Gear, Enchants, Gems, Races
- Hero Talent, Class Talent, and Spec Talent popularity breakdown
- Secondary stat priority with color-coded bars
- Best-in-Slot gear recommendations per equipment slot
- Enchant and gem recommendations with usage percentages
- Race distribution among top players
- Slash commands: `/murlok` and `/mf` to toggle panel
- `/mf debug` to toggle debug logging
- `/mf reset` to reset panel position
- `/mf help` to show available commands
- Draggable panel with position memory (SavedVariables)
- ESC key closes the panel
- Auto-hide during combat (PLAYER_REGEN_DISABLED)
- Auto-show after combat ends (PLAYER_REGEN_ENABLED)
- Localization support: English (enUS) and Simplified Chinese (zhCN)
- Data timestamp display in footer
- Scroll support for long content lists
